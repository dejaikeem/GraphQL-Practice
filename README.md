## GraphQL-Practice
Prisma, TypeScript, apollo-server, mysql를 사용하여 요구사항을 만족하는 GraphQL Server를 구현
## 레포지토리 복제
```git clone https://github.com/dejaikeem/bold9Hw.git```
## .env 설정
```DATABASE_URL="mysql://user:password@localhost:port/name"```
## 설치
```npm install```
## seed file 생성
```npx prisma db seed```
## 실행
```npm start```
